---

### Design Specifications

**Typography:**
- Title: 24-28pt, bold, dark gray (#2C3E50)
- Headings: 16-18pt, bold, accent color (#3498DB or #27AE60)
- Body: 11-12pt, regular, dark gray
- Metrics: 14-16pt, bold, accent color

**Color Palette:**
- Primary background: White (#FFFFFF)
- Snapshot box: Light blue/gray (#ECF0F1 or #E8F4F8)
- Results box: Light green (#D5F4E6) or light blue (#D6EAF8)
- Takeaway footer: Light yellow (#FEF9E7) or light gray (#F4F6F7)
- Accent text: Blue (#3498DB) or green (#27AE60)

**Spacing:**
- Margins: 0.75" all sides
- Section spacing: 12-16pt between sections
- Box padding: 12-16pt internal padding

**Visual Hierarchy:**
- Most prominent: Results box (largest text, accent background)
- Secondary: Title, Snapshot box
- Tertiary: Body sections
- Footer: Operator takeaway (distinct but subtle)

## Export Tools Recommendations

**Option 1: Markdown to PDF (Command Line)**
```bash
# Using pandoc (professional output)
pandoc CRO_CaseStudy_Segmentation_53pctLift.md -o CRO_CaseStudy_Segmentation_53pctLift.pdf --pdf-engine=xelatex

# Using markdown-pdf (simple)
markdown-pdf CRO_CaseStudy_Segmentation_53pctLift.md
```

**Option 2: Online Markdown to PDF Converters**
- [Markdown to PDF](https://www.markdowntopdf.com/) - Clean, customizable
- [CloudConvert](https://cloudconvert.com/md-to-pdf) - Professional output

**Option 3: Design Tools (Best for Premium Look)**
- **Figma** - Import markdown structure, apply custom design
- **Canva** - Use "Report" or "Case Study" templates
- **Google Docs** - Import markdown, apply custom styling, export PDF
- **Notion** - Import markdown, customize, export PDF

**Recommended:** Use Google Docs or Figma for maximum control over layout and branding.
